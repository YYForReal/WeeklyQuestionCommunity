# 学生问答社区

> 基于仿知乎项目的 二次改造 多人交流平台

## 模块

1. 用户模块
    - 登录注册
    - 信息修改
    - 个人主页（TODO）

2. 内容模块
    - 文章展示
    - 文章点赞&评论
    - 文章发布&回答

## 特色

1. jwt加密
2. 自定义markdown编辑
3. 自定义个人背景



## 项目运行流程

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report
```
